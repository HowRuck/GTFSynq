# NordSIRI

A modern application built with Spring Boot and Angular that scrapes and processes real-time rail data using the SIRI (Service Interface for Real-Time Information) standard

> **📝 Note:** This project is primarily for educational purposes, providing a platform for me to refamiliarize myself with the Spring Boot and Java development and DevOps stack. As such, the code in this repository is not primarily optimized for production, performance, or architectural best practices.
